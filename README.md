# EN-CBAM-A-Novel-Deep-Learning-Approach-for-accurate-respiratory-disease-detection
Detecting respiratory diseases like pneumonia &amp; COVID-19 using deep learning on chest X-rays. EfficientNet-B0 + CBAM model aims for 90%+ accuracy, beating existing models (80-95%) and matching complex models (89-93%). Timely diagnosis can improve healthcare outcomes!
